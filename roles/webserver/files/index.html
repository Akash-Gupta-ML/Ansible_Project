Get Current Time

Project Overview

This project demonstrates a simple web application that interacts with a three-tier architecture system. The system consists of the following components:

  • Web Server: Hosts the frontend application using Apache2. It provides a simple HTML page with a button to request the current date and time from the backend.
  • Application Server: Hosts a Flask application. It handles the request from the web server, fetches the current date and time from the database, and returns it as a JSON response.
  • Database Server: Contains a MySQL database with a table that stores and retrieves the current date and time. The Flask application queries this database to get the current time.

When the user clicks the "GET TIME NOW" button, the frontend sends a request to the Flask application running on the application server. The Flask application then queries the MySQL database on the database server to get the current date and time. This information is sent back to the frontend and displayed on the web page.
